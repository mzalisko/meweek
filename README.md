# Meweek

Особисте робоче середовище: сховище Obsidian + проєкти + AI-агенти (Claude, Codex, Gemini) в одній git-папці.

## Як відкрити

1. Obsidian → Open folder as vault → `C:\Dev\Meweek`.
2. Точка входу — `Старт.md`.

## Структура

| Папка | Призначення |
|---|---|
| `00-Inbox` | Чернетки, ідеї, щоденник |
| `10-Projects` | Активні проєкти (NMDB та інші) |
| `20-Knowledge` | Постійні знання |
| `30-Agents` | AI-агенти: профілі, конектори, навички |
| `40-Resources` | Посилання та вкладення |
| `50-Guides` | Гайди з налаштування середовища |
| `90-Archive` | Завершене |
| `_Templates` | Шаблони нотаток |

## Агенти

Спільні правила для всіх агентів — `AGENTS.md`. `CLAUDE.md` і `GEMINI.md` посилаються на нього.

Вміст нотаток і коміти — українською; папки та код — англійською.
