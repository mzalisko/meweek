# Meweek

Особисте робоче середовище: сховище Obsidian + проєкти + AI-агенти (Claude, Codex, Gemini) в одній git-папці.

## Як відкрити

1. Obsidian → Open folder as vault → `C:\Dev\Meweek`.
2. Точка входу — `Старт.md`.

## Структура

| Папка | Призначення |
|---|---|
| `00-Вхідні` | Чернетки, ідеї, щоденник |
| `10-Проєкти` | Активні проєкти (NMDB та інші) |
| `20-База знань` | Постійні знання |
| `30-Агенти` | AI-агенти: профілі, конектори, навички |
| `40-Ресурси` | Посилання та вкладення |
| `90-Архів` | Завершене |
| `_Шаблони` | Шаблони нотаток |

## Агенти

Спільні правила для всіх агентів — `AGENTS.md`. `CLAUDE.md` і `GEMINI.md` посилаються на нього.

Мова нотаток і комітів — українська.
