# Fitness — Як працювати

Інструкція для користувача: як говорити з агентом і що відбувається після кожної команди. Правила для самих агентів — [[Fitness — Команди]]; команди працюють однаково в Claude (`/fitness ...`), Codex і Gemini (`fitness ...` без слеша).

## Команди

| Команда | Що робить агент |
|---|---|
| `fitness` | показує список профілів, нічого не змінює |
| `fitness Микола` | щоденний режим: приймає звіт, оновлює журнал і профільні файли; якщо в [[Микола — Огляд]] є відкриті питання — спочатку ставить 1-2 з них |
| `fitness <ім'я> уточнення` | режим збору додаткової інформації: ставить або приймає відповіді на відкриті питання профілю без запису в журнал |
| `fitness <ім'я> input` | alias для `fitness <ім'я> уточнення` |
| `fitness Микола smoke ...` | оновлює журнал куріння у [[Микола — Звички]] |
| `fitness Марта` | onboarding: проводить базове опитування за [[Марта — Опитування]]; щоденний режим увімкнеться після завершення |
| `fitness <ім'я> підбий тиждень` | тижневий підсумок за [[Fitness — Тижневий підсумок]]: аналіз, gate, максимум один важіль корекції |
| `fitness <ім'я> імпортуй резервні звіти` | переносить ручні записи з резервного файлу профілю в журнал і профільні файли |

## Як надсилати щоденний звіт

Команда профілю + звіт одним повідомленням, формати — у [[Fitness — Звіти]]. Можна вільним текстом — агент сам розкладе дані по файлах. Мінімальний приклад:

```text
/fitness Микола
вага 82, тренування зроблено, кроки 7500, сон 5 год, харчування за планом
```

## Де дивитись що робити зараз

Відкривати персональний план дій: [[Микола — План дій]] або [[Марта — План дій]]. Це короткий файл для поточного фокусу й найближчих дій; щоденні факти лишаються в журналі профілю.

## Як закривати відкриті питання

Якщо треба просто дати додаткові дані без щоденного звіту, використовувати режим уточнень:

```text
fitness Микола уточнення
```

або одразу з відповіддю:

```text
fitness Микола input дата старту некуріння правильна, обладнання: турнік, бруси, еспандер
```

Агент оновить персональні файли й прибере закриті пункти з [[Микола — Огляд]], але не додасть запис у журнал без щоденних метрик.

## Щоденний цикл

1. Раз на день — короткий звіт агенту (1-2 хвилини).
2. Якщо був алкоголь або сильна тяга до куріння — згадати у звіті, агент рознесе по файлах.
3. Якщо день зірвався — все одно записати факт; план не обнуляється.

## Якщо агент недоступний (ліміт)

1. Відкрити резервний файл свого профілю: [[Микола — Резервні звіти]] або [[Марта — Резервні звіти]].
2. Додати рядок у таблицю зі статусом `новий`.
3. Коли агент доступний: `fitness <ім'я> імпортуй резервні звіти`.

## Щотижня

Після 5+ днів записів: `fitness Микола підбий тиждень`. Агент порахує середню вагу й виконання, звіриться з gate із [[Fitness — Принципи результативності]], змінить максимум один важіль, оновить [[Микола — План дій]] і запише рішення у [[Микола — Харчування]] / [[Микола — Тренування]], а рядок результату — у [[Fitness — Зведення]].

## Що пам'ятати

- Старт: 2026-06-13, перший тиждень — адаптаційний.
- Корекції — тільки за тижневими даними, не за одним днем.
- Межа: не шкодити здоров'ю; якщо можна простіше — робити простіше.
- Obsidian — джерело правди; календар, якщо з'явиться, — лише нагадування.
