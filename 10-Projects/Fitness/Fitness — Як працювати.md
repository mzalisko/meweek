# Fitness — Як працювати

Коротка інструкція для роботи з проєктом без пошуку по всіх нотатках.

## З чого починати

1. Відкрити [[Fitness — Контекст]].
2. Відкрити [[Fitness — Профілі]] і вибрати профіль.
3. Для Миколи відкрити [[Микола — Огляд]]; для Марти спочатку перевірити [[Марта — Огляд]] і onboarding.
4. Якщо стартує новий день для активного профілю, писати в його журнал або резервні звіти.

## Щоденний цикл

1. Перевірити вагу, кроки, тренування, сон і харчування.
2. Якщо був алкоголь, куріння або інша звичка, окремо занести це у файл відповідного профілю.
3. Наприкінці дня зробити короткий запис у журналі.
4. Якщо щось пішло не за планом, не скидати все, а просто відмітити факт.

## Якщо потрібен агент

- Для Миколи використовувати `fitness Микола`.
- Для Марти використовувати `fitness Марта`; якщо onboarding не завершено, агент має спочатку провести базове опитування.
- Якщо профіль не обрано, використовувати `fitness`, щоб побачити список.
- Для куріння використовувати `fitness Микола smoke`.
- Якщо є резервні записи, використовувати `fitness <ім'я> імпортуй резервні звіти`.

## Якщо агент недоступний

1. Відкрити резервні звіти потрібного профілю.
2. Додати запис у таблицю.
3. Поставити статус `новий`.
4. Коли агент знову доступний, написати йому про імпорт резервних звітів.

## Щотижня

1. Подивитися, чи є стійкість по вазі, талії, сну і тренуваннях.
2. Якщо треба, попросити підсумок через `fitness Микола`.
3. Не змінювати план різко, якщо є лише 1-2 дні відхилення.

## Що не забувати

- Старт уже зафіксований: від завтра.
- Формат плану комбінований.
- Микола має статус `active`; Марта має статус `onboarding_required`.
- Межа: не шкодити здоров'ю.
- Якщо щось можна зробити простіше, робити простіше.
- Якщо хочеш зручніше, календар використовувати тільки як нагадування, а Obsidian лишити джерелом правди.
