# Марта — Огляд

Персональний дашборд Марти в межах [[Fitness — Огляд|Fitness]]. Поки активна робота ведеться з [[Микола — Огляд|Миколою]], ця нотатка зберігає структуру для майбутнього старту Марти.

## Швидкий стан

| Напрям | Стан |
|---|---|
| Опитування | [[Марта — Опитування]] не заповнено |
| Параметри | [[Марта — Параметри]] не заповнено |
| Харчування | [[Марта — Харчування]] очікує параметрів |
| Тренування | [[Марта — Тренування]] очікує параметрів |
| Журнал | [[Журнал — Марта]] |
| Звіти | [[Fitness — Звіти]] |

## Наступний крок

Почати тільки після окремої команди користувача:

```text
Починаємо Fitness для Марти.
```

Після цього агент має заповнити [[Марта — Опитування]] і перенести стислий підсумок у [[Марта — Параметри]].
