# Марта — Огляд

Персональний дашборд Марти в межах [[Fitness — Огляд|Fitness]]. Профіль має статус `onboarding_required` у [[Fitness — Профілі]], тому щоденний режим не запускати до завершення базового опитування.

## Швидкий стан

| Напрям | Стан |
|---|---|
| Опитування | [[Марта — Опитування]] не заповнено |
| Параметри | [[Марта — Параметри]] не заповнено |
| Харчування | [[Марта — Харчування]] очікує onboarding |
| Тренування | [[Марта — Тренування]] очікує onboarding |
| Сон | [[Марта — Сон]] очікує onboarding |
| Звички | [[Марта — Звички]] очікує onboarding |
| Алкоголь | [[Марта — Алкоголь]] очікує onboarding |
| Журнал | [[Журнал — Марта]] |
| Резервні звіти | [[Марта — Резервні звіти]] |
| Звіти | [[Fitness — Звіти]] |

## Наступний крок

Почати через команду:

```text
fitness Марта
```

Після цього агент має провести onboarding: заповнити [[Марта — Опитування]], перенести стислий підсумок у [[Марта — Параметри]], [[Марта — Сон]], [[Марта — Звички]] і [[Марта — Алкоголь]], потім створити стартові [[Марта — Харчування]] і [[Марта — Тренування]].
