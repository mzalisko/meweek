# Fitness — Звіти

Це короткі формати повідомлень для ведення [[Fitness — Огляд|Fitness]]. Їх можна надсилати агенту без додаткових пояснень.

## Щоденний звіт

```text
Fitness звіт:
учасник: Микола / Марта
дата: 2026-06-11
вага: __ кг
тренування: зроблено / пропущено / не було за планом
кроки: __
харчування: за планом / частково / ні
сон: __ год
голод: 1-10
енергія: 1-10
алкоголь: ні / напій + обсяг / коктейль + склад
нотатка: __
```

Агент після такого звіту:

1. додає рядок у [[Микола — Журнал]] або [[Марта — Журнал]];
2. не змінює калорійність за одним днем;
3. якщо є біль, сильна втома або різкий стрибок ваги — позначає це в нотатці;
4. якщо це кінець тижня — пропонує підсумок за [[Fitness — Тижневий підсумок]].

## Швидкий звіт

```text
Fitness: Микола, 2026-06-11, вага __, тренування зроблено/пропущено, кроки __, нотатка __.
```

## Алкогольний запис

```text
Алкоголь Fitness:
дата: 2026-06-11
напій: __
обсяг: __
коктейль/чистий: __
привід: __
їжа поруч: __
сон після: __
самопочуття: __
чи повторити: так/ні/змінити
нотатка: __
```

Агент додає такий запис у [[Микола — Алкоголь]] і, якщо потрібно, коротко оновлює матрицю у [[Микола — Звички]].

## Стартові відповіді

Коли учасник відповів на опитування, агент:

1. стисло переносить дані у персональний файл параметрів учасника: [[Микола — Параметри]] або [[Марта — Параметри]];
2. не зберігає зайві приватні деталі, якщо вони не потрібні для плану;
3. розраховує тільки орієнтовні КБЖВ і позначає їх як не медичну пораду;
4. оновлює [[Fitness — Контекст]].

## Команди для агента

```text
Прочитай контекст Fitness і додай цей щоденний звіт у правильний журнал.
```

```text
Підбий тижневий підсумок Fitness за журналами і запропонуй корекції без різких змін.
```
