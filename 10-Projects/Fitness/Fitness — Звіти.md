# Fitness — Звіти

Це короткі формати повідомлень для ведення [[Fitness — Огляд|Fitness]]. Їх можна надсилати агенту без додаткових пояснень.

## Щоденний звіт

```text
Fitness звіт:
учасник: Микола / Марта
дата: 2026-06-11
вага: __ кг
тренування: зроблено / пропущено / не було за планом
кроки: __
харчування: за планом / частково / ні
сон: __ год
голод: 1-10
енергія: 1-10
алкоголь: ні / напій + обсяг / коктейль + склад
виконання плану: 0-100% або коротко
нотатка: __
```

Агент після такого звіту:

1. додає рядок у [[Журнал — Микола]] або [[Журнал — Марта]]; голод, енергію, виконання плану й інші поля без власної колонки стисло заносить у «Нотатка»;
2. оновлює профільні файли тільки для вказаного учасника;
3. не змінює калорійність за одним днем;
4. якщо є біль, сильна втома або різкий стрибок ваги — позначає це в нотатці;
5. якщо є сон, алкоголь або звички — оновлює відповідні персональні файли;
6. якщо це кінець тижня — пропонує підсумок за [[Fitness — Тижневий підсумок]] і правилами [[Fitness — Принципи результативності]].

## Швидкий звіт

```text
Fitness: Микола, 2026-06-11, вага __, тренування зроблено/пропущено, кроки __, нотатка __.
```

## Якщо агент недоступний

Якщо Claude/Codex недоступний через тижневий ліміт або інше обмеження, звіт не треба тримати в голові чи в чаті. Для кожного профілю є окремий резервний файл: [[Микола — Резервні звіти]] або [[Марта — Резервні звіти]].

Мінімальний ручний запис:

```text
| 2026-06-11 | 82 кг | зроблено | 7000 | частково | 5 год | пиво 1 л | 0 | коротка нотатка | новий |
```

Коли агент знову доступний:

```text
fitness <ім'я> імпортуй резервні звіти
```

## Алкогольний запис

```text
Алкоголь Fitness:
дата: 2026-06-11
напій: __
обсяг: __
коктейль/чистий: __
привід: __
їжа поруч: __
сон після: __
самопочуття: __
чи повторити: так/ні/змінити
нотатка: __
```

Агент додає такий запис у персональний файл алкоголю: [[Микола — Алкоголь]] або [[Марта — Алкоголь]]. Якщо потрібно, коротко оновлює відповідну матрицю звичок.

## Стартові відповіді

Коли учасник відповів на опитування, агент:

1. стисло переносить дані у персональний файл параметрів учасника: [[Микола — Параметри]] або [[Марта — Параметри]];
2. переносить сон, звички й алкоголь у персональні файли, якщо ці дані вже є;
3. не зберігає зайві приватні деталі, якщо вони не потрібні для плану;
4. розраховує тільки орієнтовні КБЖВ і позначає їх як не медичну пораду;
5. оновлює [[Fitness — Профілі]] і [[Fitness — Контекст]].

## Команди для агента

```text
Прочитай контекст Fitness і додай цей щоденний звіт у правильний журнал.
```

```text
Підбий тижневий підсумок Fitness за журналами і запропонуй корекції без різких змін.
```
