# Fitness — Огляд

## Мета

Схуднення через розраховане меню + програму вправ, з веденням журналу результатів. Система підтримує кілька профілів через [[Fitness — Профілі]]: [[Микола — Огляд|Микола]] активний, [[Марта — Огляд|Марта]] очікує базового опитування. У кожного профілю власна папка з планом дій, параметрами, харчуванням, тренуваннями, сном, звичками, алкоголем, журналом і резервними звітами.

## Як користуватись (будь-який агент: Claude / Codex / Gemini)

Повна інструкція для користувача — [[Fitness — Як працювати]]. Коротко:

1. Щоденний звіт: `fitness Микола` (Codex/Gemini) або `/fitness Микола` (Claude) + текст звіту за [[Fitness — Звіти]].
2. Уточнення даних без журналу: `fitness Микола уточнення` або `fitness Микола input ...`.
3. Куріння: `fitness Микола smoke ...`.
4. Раз на тиждень: `fitness Микола підбий тиждень` — агент аналізує динаміку й коригує максимум один важіль.

## Важливо

Розрахунки агента — орієнтовні, не медична порада. При хронічних станах чи різких змінах ваги — консультація лікаря.

Файли проєкту містять персональні дані про здоров'я і звички: репозиторій тримати приватним, нові чутливі деталі додавати лише якщо вони потрібні для плану.
