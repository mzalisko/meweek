# План імплементації 2.5 — Bulk-дії над виділеними значеннями

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development. Steps use checkbox (`- [ ]`).

**Goal:** Виділяти кілька рядків гріда чекбоксами й застосовувати масову дію з **прев'ю «торкнеться N сайтів»** перед застосуванням. Перша масова дія — **видалення** виділених значень (із підтвердженням + публікацією всіх уражених сайтів одним рухом). Це закладає інфраструктуру виділення+прев'ю, яку перевикористають інші bulk-дії.

**Architecture:** Розширюємо `ValuesGrid`: стан `selected[]` (id значень), чекбокси в рядках (замість плейсхолдера ☐), смужка bulk-дій угорі при `count(selected)>0` з лічильником «N виділено · торкнеться M сайтів». Прев'ю M = к-сть **унікальних** сайтів, уражених усіма виділеними значеннями (об'єднання `AffectedSites::for` по кожному). `bulkDelete()` — зняти знімок уражених сайтів ДО видалення, видалити `DataValue` виділених, аудит пакета (`bulk.deleted` з переліком ключів), опублікувати кожен унікальний уражений сайт **один раз**, очистити виділення.

**Tech Stack:** Laravel, Livewire 3, Tailwind, PHPUnit + Livewire::test, Http::fake.

**Спека:** `DBManager — Дизайн.md` §7 (bulk-дії з прев'ю «торкнеться N сайтів»), §10 (помилковий bulk → відкат), §11 (аудит пакетом). Макет `grid-failover-v5.html`: рядок фільтрів має «☑ 2 → Bulk-дії ▾».

**Сервіси/моделі:** `App\Admin\AffectedSites::for(DataValue): Collection<Site>` (з 2.3). `App\Models\DataValue`. `App\Services\Publishing\SitePayloadCompiler::publish(Site)`, `App\Services\Publishing\BridgePublisher::push(Publication)`. `App\Models\AuditLog`. `App\Livewire\ValuesGrid` (наявний; рядки мають `id`=dataValue id, `type`).

namespace Tests\Feature\Admin;

use App\Models\Site;

use Illuminate\Foundation\Testing\RefreshDatabase;
use Livewire\Livewire;

class GridSelectionTest extends TestCase
{
    use RefreshDatabase;

    protected function setUp(): void
    {
        parent::setUp();
        $this->actingAs(User::factory()->create());
    }

    public function test_toggle_select_adds_and_removes_ids(): void
    {
        $site = Site::factory()->create();
        $a = DataValue::factory()->forSite($site)->create(['key' => 'a']);
        $b = DataValue::factory()->forSite($site)->create(['key' => 'b']);

        Livewire::test(ValuesGrid::class, ['site' => $site->id])
            ->call('toggleSelect', $a->id)
            ->assertSet('selected', [$a->id])
            ->call('toggleSelect', $b->id)
            ->assertSet('selected', [$a->id, $b->id])
            ->call('toggleSelect', $a->id)
            ->assertSet('selected', [$b->id]);
    }

    public function test_clear_selection(): void
    {
        $site = Site::factory()->create();
        $a = DataValue::factory()->forSite($site)->create(['key' => 'a']);

        Livewire::test(ValuesGrid::class, ['site' => $site->id])
            ->call('toggleSelect', $a->id)
            ->call('clearSelection')
            ->assertSet('selected', []);
    }
}

```php
public function test_bulk_delete_removes_values_audits_and_publishes(): void
{
    config(['services.bridge.ingest_url'=>'https://bridge.local/api/internal/publish','services.bridge.publish_secret'=>'pub']);
    \Illuminate\Support\Facades\Http::fake(['*'=>\Illuminate\Support\Facades\Http::response(['stored_version'=>1],200)]);
    $this->actingAs(\App\Models\User::factory()->create());
    $site = \App\Models\Site::factory()->create(['domain'=>'domen.ua']);
    app(\App\Services\Provisioning\SiteProvisioner::class)->issueToken($site);
    $a = \App\Models\DataValue::factory()->forSite($site)->create(['key'=>'a']);
    $b = \App\Models\DataValue::factory()->forSite($site)->create(['key'=>'b']);

    \Livewire\Livewire::test(\App\Livewire\ValuesGrid::class, ['site'=>$site->id])
        ->set('selected', [$a->id, $b->id])
        ->call('bulkDelete')
        ->assertSet('selected', []);

    $this->assertSame(0, \App\Models\DataValue::whereIn('id',[$a->id,$b->id])->count());
    $this->assertTrue(\App\Models\AuditLog::where('action','bulk.deleted')->exists());
    \Illuminate\Support\Facades\Http::assertSentCount(1); // domen.ua опубліковано один раз
}
```
