# План імплементації 2.6 — Єдина бічна панель редагування

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development. Steps use checkbox (`- [ ]`).

**Goal:** Клік на будь-який не-телефонний рядок → відкривається права бічна панель 420px (замість модалки). У панелі: редагувати значення, **гео-мітки** (toggle-чіпи), **прив'язку месенджера до слота** (`linked_slot`), область дії, Зберегти/Видалити. Кнопка «+ Додати» — теж відкриває цю панель у режимі CREATE. Взаємне виключення: тільки одна панель відкрита за раз (відкриття однієї закриває іншу). Стиль — синій акцент `#54708c`, Lucide-іконки.

**Architecture:**
- `ValueEditor` (Livewire) перестає бути overlay-модалкою → стає фіксованою правою панеллю: `fixed top-0 right-0 bottom-0 z-40 w-[420px]` (overlay поверх пустого slot-panel aside).
- Взаємне виключення через Livewire-події: `close-slot-panel` (ValueEditor → SlotPanel) та `close-editor-panel` (SlotPanel → ValueEditor).
- Гео-мітки: `$geoTagIds: array` у Livewire; `GeoTag::orderBy('code')->get()` в рендері; `geoTags()->sync()` при збереженні.
- linked_slot: `$linkedSlot: string` у Livewire; при типі `messenger` → показати select з доступними phone-ключами того ж scope.

**Tech Stack:** Laravel, Livewire 3, Tailwind, PHPUnit + Livewire::test.

**Спека:** `DBManager — Дизайн.md` §7 (панель слота праворуч, без виходу зі списку), §4 (гео-мітки на DataValue, прив'язані месенджери follow slot).

**Сервіси/моделі:** `App\Livewire\ValueEditor`, `App\Livewire\SlotPanel`, `App\Models\{DataValue,GeoTag}`, `App\Admin\AffectedSites`, `App\Services\Publishing\{SitePayloadCompiler,BridgePublisher}`.

```php
<?php

namespace Tests\Feature\Admin;

use App\Livewire\ValueEditor;
use App\Models\DataValue;
use App\Models\GeoTag;
use App\Models\Site;
use App\Models\User;
use Illuminate\Foundation\Testing\RefreshDatabase;
use Livewire\Livewire;
use Tests\TestCase;

class ValueEditorGeoTest extends TestCase
{
    use RefreshDatabase;

    protected function setUp(): void
    {
        parent::setUp();
        $this->actingAs(User::factory()->create());
    }

    public function test_edit_loads_existing_geo_tags(): void
    {
        $site  = Site::factory()->create();
        $geo   = GeoTag::create(['code' => 'UA', 'name' => 'Україна']);
        $dv    = DataValue::factory()->forSite($site)->create(['key' => 'k']);
        $dv->geoTags()->attach($geo->id);

        Livewire::test(ValueEditor::class)
            ->call('edit', $dv->id)
            ->assertSet('geoTagIds', [$geo->id]);
    }

    public function test_save_syncs_geo_tags(): void
    {
        $site  = Site::factory()->create();
        $ua    = GeoTag::create(['code' => 'UA', 'name' => 'Україна']);
        $world = GeoTag::create(['code' => 'WORLD', 'name' => 'Світ']);
        $dv    = DataValue::factory()->forSite($site)->create(['key' => 'k']);
        $dv->geoTags()->attach($ua->id);

        Livewire::test(ValueEditor::class)
            ->call('edit', $dv->id)
            ->set('geoTagIds', [$world->id])
            ->call('save');

        $this->assertSame([$world->id], $dv->fresh()->geoTags->pluck('id')->all());
    }

    public function test_save_creates_audit_for_geo_change(): void
    {
        $site  = Site::factory()->create();
        $ua    = GeoTag::create(['code' => 'UA', 'name' => 'Україна']);
        $dv    = DataValue::factory()->forSite($site)->create(['key' => 'k']);

        Livewire::test(ValueEditor::class)
            ->call('edit', $dv->id)
            ->set('geoTagIds', [$ua->id])
            ->call('save');

        $this->assertTrue(\App\Models\AuditLog::where('action', 'value.geo_changed')->exists());
    }
}
```

```php
    {
        $site = Site::factory()->create();

            'key'     => 'm',
            'content' => ['value' => 'https://t.me/x', 'network' => 'telegram', 'linked_slot' => 'phone_ua_1'],

            ->call('edit', $dv->id)

    public function test_save_messenger_persists_linked_slot(): void
    {
        $site = Site::factory()->create();

            'key'           => 'phone_ua_1',
            'value_type_id' => $phoneType->id,
            'content'       => [],

        \App\Models\PhoneSlot::create(['data_value_id' => $slot->id, 'return_mode' => 'auto', 'exhaustion_policy' => 'hide']);

            'key'           => 'm',

            ->call('save');
```
