# План імплементації 2.4 — Телефонний слот і керування номерами

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development. Steps use checkbox (`- [ ]`).

**Goal:** Створювати телефонне значення (слот) із адмінки й керувати його ланцюгом номерів у панелі слота: додати номер, видалити, переставити пріоритет (вгору/вниз). Це закриває «як додати телефон для сайта» — досі телефонні слоти можна було лише редагувати (2.2) і сидіти кодом, а не створювати/наповнювати через UI.

**Architecture:** Розширюємо два наявні Livewire-компоненти. `ValueEditor` (2.3) отримує тип **phone**: створення phone-значення робить `DataValue(phone)` + `PhoneSlot` (return_mode=auto, exhaustion_policy=hide), після чого відкриває панель слота (`open-slot`), щоб одразу додавати номери. `SlotPanel` (2.2) отримує керування номерами: `addNumber` (нове `PhoneNumber` + `NumberEntry` з наступним пріоритетом), `removeNumber`, `moveUp`/`moveDown` (своп пріоритету з сусідом). Після кожної зміни — `FailoverEngine::recompute` (перерахунок показуваного номера) + аудит. Публікація в bridge — наявною кнопкою «Зберегти» панелі (зміни застосовуються в БД одразу, у bridge ідуть на Save; модель «застосувати зараз, опублікувати на Save» — як у 2.2).

**Tech Stack:** Laravel, Livewire 3, Tailwind, PHPUnit + Livewire::test.

**Спека:** `DBManager — Дизайн.md` §4 (PhoneSlot/NumberEntry, пріоритети #0 основний/#1.. резерви, failover). Макет `grid-failover-v5.html` секція `.panel`: «+ додати резерв», grip-реордер, статуси.

**Моделі/сервіси:** `App\Models\PhoneSlot` (data_value_id, return_mode, exhaustion_policy, pinned_number_entry_id, current_number_entry_id). `App\Models\NumberEntry` (phone_slot_id, phone_number_id, priority; relations slot/phoneNumber; `$guarded=[]`). `App\Models\PhoneNumber` (e164, label, status['active'|'down'], down_since; `$guarded=[]`). `App\Models\DataValue` (type phone via ValueType code 'phone'). `App\Services\Failover\FailoverEngine::recompute(PhoneSlot,'user')`. `App\Models\AuditLog::create([...])`. `App\Livewire\{ValueEditor,SlotPanel}` (наявні).

    public function test_creating_phone_value_creates_slot_and_opens_panel(): void

            ->call('createFor', $site->id)
            ->set('type', 'phone')

        $this->assertSame('auto', $slot->return_mode);

        $this->actingAs(User::factory()->create());

            ->call('createFor', $site->id)
            ->set('type', 'phone')->set('key', 'phone_x')->set('scope', 'site')
            ->call('save')
            ->assertHasNoErrors();

}

public function test_add_number_creates_entry_at_next_priority(): void
{
    $this->actingAs(\App\Models\User::factory()->create());
    $site = \App\Models\Site::factory()->create();
    [$slot] = $this->slotWithNumbers(['active']); // 1 номер, priority 0
    $slot->dataValue->update(['scope_type'=>'site','scope_id'=>$site->id]);

    \Livewire\Livewire::test(\App\Livewire\SlotPanel::class)
        ->call('open', $slot->dataValue->id)
        ->set('newNumber', '+380441112233')
