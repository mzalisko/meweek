# DBManager — Контекст

**Оновлено:** 2026-06-19
**Стан:** Впроваджено комплекс покращень UI/UX для сторінки масових операцій (`/admin/bulk`). Усі тести проходять.

## Актуально
- Код проєкту: `10-Projects/DBManager/code/`.
- Додано Livewire-сторінку **Масові операції**: `/admin/bulk`, пункт навігації веде туди.
- Впроваджено **інтерактивний Diff-preview**: у таблиці прев'ю відображаються майбутні зміни значень, ключів, гео-тегів та стану (із кольоровим підсвічуванням та перекресленням старих значень).
- Замінено `select` області (`scope`) на **інтерактивні таби-картки** з Lucide-іконками та лічильником вибраних сайтів.
- Додано **кнопку швидкого скидання** всіх фільтрів цілей.
- Додано **індикатори завантаження (`wire:loading`)** для списку сайтів та таблиці цілей.
- Реалізовано іконки Lucide для кожного типу даних у таблиці прев'ю.
- Додано копіювання Batch UUID в один клік та пряме посилання на журнал аудиту за цим Batch UUID.
- Людський опис оновлено в [[Гайд — Адмінка DBManager (CRM)]].

## Перевірено локально
- `docker compose exec -T core php -l app/Livewire/BulkOperations.php` — синтаксис OK.
- `docker compose exec -T core php artisan test tests/Feature/Admin/BulkOperationsTest.php` — 6 тестів, 26 assertions PASS.
- Browser-перевірка `/admin/bulk`: робота табів-карт, diff-preview, скидання фільтрів, loading overlay, копіювання UUID та перехід до аудиту.
