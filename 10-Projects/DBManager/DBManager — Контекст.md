# DBManager — Контекст

**Оновлено:** 2026-06-18
**Стан:** Виконано перехід на локальні дані сайтів без наслідування від предків (сателіти є суто структурними зв'язками, динамічне наслідування ієрархії вилучено).

## Ключові зміни
- **Брендинг:** Інтегровано новий бренд системи — **DataBridge Core** (назва у хедері, заголовках та файлах конфігурації).
- **Візуальні оновлення:** Повернуто оригінальний логотип-конектор, адаптовано його під колір системи (`text-[#69bf81]`) та збільшено на 20%. Оновлено логотипи у шапці адмінки та на сторінці входу.
- **Компоненти UI:** Переписано `<select>` у хлібних крихтах на кастомні випадаючі списки на Alpine.js. Уніфіковано стиль хлібних крихт (плашки-бейджи).
- **Виправлення помилок:** Виправлено `Duplicate entry` в `AuditRestorer.php` через `updateOrCreate`. Додано відступи для таблиць в історії змін.
- **Стилі гріда:** Усунено проблему з відображенням гріда значень (ширина останньої колонки `68px`) шляхом перекомпіляції Tailwind CSS (`npm run build`) та очищення кешів (`view:clear`, `cache:clear`).
- **Локальні дані:** Кожен сайт тепер зберігає та читає власні значення (`scope_type = 'site'`). Поняття групового scope (`scope_type = 'group'`) та динамічного наслідування/перекриття значень повністю видалено.
- **Клонування при приєднанні:** У `SitesManager` додано кнопку «Клонувати дані з джерела» (з `wire:confirm` підтвердженням). Вона дозволяє менеджеру вручну одноразово скопіювати всі дані з обраного батьківського сайту-джерела на дочірній сайт, якщо на дочірньому ще немає власних значень.
- **Спрощення коду:** Вилучено застарілі класи `ValueScope.php`, `PhoneSlotInheritance.php`. Заглушено трейт `HandlesScopeDecision.php` для сумісності.
- **UI зміни:** З панелей редагування (`ValueEditor`, `SlotPanel`) та гріда вилучено вибір області дії (колонку «Область», радіо-перемикачі «Група/Цей сайт» та діалог вибору).
- **Сідер:** Оновлено `DemoDataSeeder.php` для засівання значень безпосередньо в окремі сайти (`scope_type = 'site'`) замість `'group'`.
- **Тести:** Всі 234 тести проєкту повністю адаптовані та проходять успішно (зелені).
