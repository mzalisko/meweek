# Claude Code — Dynamic Workflows

## Що це

Dynamic Workflows — можливість Claude Code (Opus 4.8) автоматично генерувати orchestration-скрипти, розбивати велику задачу на підзадачі та запускати сотні паралельних субагентів в одній сесії.

На відміну від ручного налаштування команд агентів, Claude сам визначає структуру роботи на основі цілі.

## Чому важливо

Закриває клас задач, які раніше були «надто великими для одного агента»:
- Міграція сотень тисяч рядків коду від старт до merge
- Security audit великих кодових баз
- Дослідження widespread bugs з аналізом симптомів
- Архітектурний аналіз складних систем

## Як працює

1. Користувач задає ціль на рівні проєкту
2. Claude (оркестратор) будує план і генерує orchestration script
3. Запускаються десятки/сотні субагентів паралельно на спільній файловій системі
4. Оркестратор верифікує результати субагентів
5. Зведений результат повертається користувачу

## Приклад

```
Проведи security audit всього бекенду. Перевір автентифікацію, авторизацію, 
SQL-запити та зовнішні виклики. Дай звіт критичних вразливостей.
```

Claude розіб'є це на: аудит auth модуля, аудит DB-запитів, аудит API-виклики, аудит middleware — паралельно.

## Як використати в Meweek

- Великі рефакторинги (наприклад, перехід на нову структуру папок)
- Аналіз усіх проєктів у сховищі для перехресних посилань
- Масова перевірка дотримання правил AGENTS.md у всіх проєктних нотатках
