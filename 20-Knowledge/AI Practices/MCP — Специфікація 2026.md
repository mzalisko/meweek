# MCP — Специфікація 2026

## Що це

Model Context Protocol (MCP) — відкритий стандарт для підключення AI-агентів до зовнішніх інструментів і даних. У 2026 виходить Release Candidate найбільшої ревізії протоколу — запланований реліз 28 липня 2026.

## Ключові зміни у версії 2026

### Stateless Core
- Remote MCP-сервер тепер може бути за звичайним round-robin load balancer
- Маршрутизація по `Mcp-Method` заголовку замість sticky sessions
- Клієнти можуть кешувати `tools/list` відповіді на час `ttlMs`

### MCP Apps
- Server-rendered UI через MCP — сервер може повертати живі інтерфейси
- Відкриває шлях до embedded UX у Claude Code і Desktop

### Tasks Extension
- Підтримка довготривалих асинхронних операцій
- Агент запускає задачу, отримує ID, перевіряє статус пізніше
- Ідеально для batch-операцій і фонових процесів

### Оновлений Auth
- Вирівнювання з OAuth 2.0 і OpenID Connect
- `claude mcp login` / `claude mcp logout` — CLI-автентифікація без інтерактивного меню

## Екосистема (середина 2026)

- **500+ публічних серверів**: GitHub, Slack, PostgreSQL, Stripe, Figma, Docker, Kubernetes...
- **SDK**: TypeScript, Python, C#, Java, Swift
- **Enterprise adoption**: Zendesk (MCP Server — літо 2026)

## Як можу використати

- **`claude mcp login github`** — один раз автентифікувати GitHub MCP замість `/mcp` меню
- **Tasks extension** — для рутин, де зараз потрібні poll-цикли (daily-digest, weekly-review)
- **Stateless deploy** — при переході до self-hosted MCP для Obsidian
