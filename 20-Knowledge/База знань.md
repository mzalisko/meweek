# 📚 База знань

Постійні, перевірені знання — те, що залишається корисним незалежно від проєктів.

## Структура

Нотатки організовуються плоско, навігація — через теги та посилання. Рекомендовані теги:

- `#wordpress` — плагіни, хуки, REST API, безпека WP
- `#php`, `#js`, `#python` — мови та інструменти
- `#безпека` — автентифікація, шифрування, hardening
- `#інфраструктура` — сервери, деплой, бекапи
- `#рішення` — ADR (архітектурні рішення, шаблон: [[_Templates/Шаблон ADR]])
