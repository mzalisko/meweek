# DBManager — Шорткоди ціни та телефону

Пов'язано з [[DBManager — Огляд]].

## 1. Універсальний шорткод ціни `[dbm_price]`

Шорткод `[dbm_price]` динамічно відображає правильну ціну та її валюту/одиницю вимірювання (лейбл) відповідно до гео-позиції відвідувача (визначеної по IP або через активну геосимуляцію).

### Параметри

* `key` (обов'язковий) — унікальний ключ цінового слота (наприклад, `romania`).
* `show_label` (опціонально, за замовчуванням `yes`) — чи виводити валюту/лейбл після ціни (`yes` або `no`).
* `tag` (опціонально, за замовчуванням `span`) — HTML-тег, в який буде загорнуто ціну (наприклад: `span`, `div`, `strong`, `p`, `b`).
* `class` (опціонально) — додаткові CSS-класи для оформлення HTML-тегу.

### Приклади використання

1. **Базове виведення:**
   ```text
   [dbm_price key="romania"]
   ```
   * *Для відвідувача з України покаже:* `<span>100 грн</span>`
   * *Для відвідувача з Румунії покаже:* `<span>15 RON</span>`
   * *Для інших країн (WORLD) покаже:* `<span>5 EUR</span>`

2. **Виведення тільки числового значення (без валюти):**
   ```text
   [dbm_price key="romania" show_label="no"]
   ```
   * *Покаже:* `<span>100</span>`

3. **Стилізоване виведення:**
   ```text
   [dbm_price key="romania" tag="strong" class="highlighted-price"]
   ```
   * *Покаже:* `<strong class="highlighted-price">100 грн</strong>`

## 2. Шорткод блоку телефону `[dbm_phone_block]`

Шорткод `[dbm_phone_block]` виводить номер телефону (разом із зручною іконкою трубки) та всі прикріплені до нього месенджери (Telegram, WhatsApp, Viber) у вигляді єдиного блоку, який можна гнучко стилізувати.

### Параметри

* `key` (обов'язковий) — унікальний ключ слота телефону (наприклад, `main_phone`).
* `layout` (опціонально, за замовчуванням `row`) — напрямок розташування іконок месенджерів відносно телефону (`row` — в рядок поруч, `column` — в колонку під телефоном).
* `class` (опціонально) — додаткові CSS-класи для всього контейнера.

### Приклади використання

1. **Розташування в рядок (поруч):**
   ```text
   [dbm_phone_block key="main_phone" layout="row"]
   ```

2. **Розташування в колонку (одне під одним):**
   ```text
   [dbm_phone_block key="main_phone" layout="column" class="sidebar-contacts"]
   ```

## 3. Гео-обмеження та виключення

Обидва шорткоди повністю сумісні з логікою гео-дозволів та гео-виключень:
* Якщо поточна країна відвідувача внесена в заборонений список конкретного слота (наприклад, `!RU` або `!UA`), то шорткод поверне порожній рядок.
* Якщо месенджер прикріплений до телефону, але заборонений для поточної країни (наприклад, `!UA`), то сам телефон виведеться, але іконка цього конкретного месенджера поруч із ним буде прихована.
* За симуляції іншої країни (через сторінку **Геосимуляція** в адмінці) відображення адаптується миттєво під обрану країну.
