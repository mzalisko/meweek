# Гайд — Як починати проєкт

Коли ідея вже не просто нотатка, а окрема робота, її треба заводити як проєкт.

## Як сказати агенту

```text
створи проєкт <назва>
```

або в Claude:

```text
/new-project <назва>
```

## Що має з'явитися

- папка `10-Projects/<Назва>/`;
- огляд проєкту;
- план;
- короткий контекст для наступних сесій;
- запис у списку проєктів.

## Як перевірити

1. Відкрити `10-Projects/<Назва>/`.
2. Перевірити, що є файл `<Назва> — Огляд.md`.
3. Перевірити, що в огляді є мета, ключові файли й наступний крок.
