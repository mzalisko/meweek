# Гайд — Команди для агентів

Власні **слеш-команди** Claude Code живуть у `.claude/commands/` — кожен `.md` файл стає командою `/<назва-файлу>`. Це швидкі «кнопки» поверх навичок: команда викликається явно, навичка — спрацьовує сама за контекстом.

## Наші команди (вже створені)

| Команда | Що робить |
|---|---|
| `/new-project <Назва>` | Створює проєкт у `10-Projects/` за навичкою new-project |
| `/save-context` | Зберігає контекст сесії у `<Проєкт> — Контекст.md` перед /clear |
| `/weekly-review` | Розбір 00-Inbox, перевірка контекстів, архівація |

## Як створити нову команду

1. Створити файл `.claude/commands/<назва>.md` (назва — латиницею, через дефіс).
2. Вміст — інструкція агенту українською. `$ARGUMENTS` — підставляє аргументи виклику.

Приклад `.claude/commands/adr.md`:

```markdown
Оформи технічне рішення як ADR у папці поточного проєкту
за шаблоном _Templates/Шаблон ADR.md. Тема рішення: $ARGUMENTS
```

3. Команда одразу доступна в сесії як `/adr <тема>`.

**Можна доручити агенту:** «створи команду /<назва>, яка робить <що>».

## Де що описувати

- Команди → цей гайд + файли в `.claude/commands/`
- Навички → [[30-Agents/Конектори і навички]] + `.claude/skills/`
