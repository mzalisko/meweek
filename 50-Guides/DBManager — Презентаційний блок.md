# DBManager — презентаційний блок

Пов'язано з [[DBManager — Огляд]].

## Що це

Презентаційний блок плагіна показує дані, які вже доставлені з CRM у кеш WordPress: номери, месенджери й ціни. Він не робить зовнішніх запитів і не відкриває нових точок входу.

## Як вставити

У сторінку або запис WordPress вставити:

```text
[dbm_presentation]
```

Короткий alias:

```text
[dbm_block]
```

Опційний заголовок:

```text
[dbm_presentation title="domen.ua"]
```

## Як перевірити

1. У DBManager Core змінити дані сайту й дочекатися доставки в плагін.
2. Відкрити сторінку з блоком.
3. Перевірити вкладки “Усе”, “Номери”, “Месенджери”, “Ціни”, пошук і згортання секцій.

Блок бере порядок із кешу плагіна: телефони → месенджери → ціни. Для телефонів показує форматований текст із CRM, якщо для слота задано шаблон.

Статуси, типи значень, гео й основні дані виділені в стилі CRM, щоб блок можна було використовувати для презентації без додаткового дизайну.
