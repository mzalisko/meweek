# Codex

**Конфіг:** `AGENTS.md` (корінь) — стандартний файл інструкцій, який Codex читає автоматично. Папка `.codex/` — для локальної конфігурації; глобальний конфіг — `~/.codex/config.toml`.

## Встановлення

```
npm install -g @openai/codex
codex            # запустити в C:\Dev\Meweek
```

## Для чого використовувати

- Кодові задачі з терміналу: написання, виправлення, тести
- Швидкі правки у проєктах
- Паралельна робота: Codex над кодом, Claude над документацією/архітектурою

## Особливості

- Читає `AGENTS.md` з кореня та з підпапок (ближчий файл має пріоритет — можна класти окремий `AGENTS.md` у папку проєкту з кодом).
- Режими доступу: read-only / auto / full-access — для цього сховища рекомендований **auto**.
