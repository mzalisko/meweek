# Конектори і навички

Що підключено до Claude (Cowork) у цьому середовищі та як цим користуватися.

## Підключені конектори (станом на 2026-06-11)

| Конектор | Можливості |
|---|---|
| GitHub (плагін engineering) | Робота з репозиторіями, PR, issues |
| Gmail | Пошук листів, чернетки, мітки |
| Notion | Сторінки, бази даних |
| Figma | Читання/створення дизайнів, діаграми FigJam |
| Google Drive | Пошук і читання файлів |
| Slack, Linear, Jira/Confluence, Datadog, PagerDuty, Asana | Доступні через плагін engineering (потребують автентифікації) |
| Claude in Chrome | Керування браузером |
| Computer use | Керування десктопними застосунками |

## Навички (skills)

| Навичка | Для чого |
|---|---|
| docx / pdf / pptx / xlsx | Створення й редагування документів |
| engineering:architecture | ADR — архітектурні рішення |
| engineering:system-design | Проєктування систем (знадобиться для NMDB) |
| engineering:code-review | Рев'ю коду перед merge |
| engineering:security-review | Безпековий аналіз змін |
| engineering:documentation | Технічна документація, README, runbook |
| engineering:testing-strategy | Стратегії тестування |
| schedule | Заплановані задачі (щоденні звіти тощо) |
| skill-creator | Створення власних навичок |

## Як користуватися

- Конектори вмикаються в налаштуваннях Claude (Settings → Connectors).
- Навички — Settings → Capabilities; спрацьовують автоматично за контекстом запиту.
- Для NMDB найкорисніші: `system-design`, `architecture`, `security-review` + конектор GitHub.
