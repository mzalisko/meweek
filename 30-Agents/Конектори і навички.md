# Конектори і навички

Що підключено до Claude (Cowork) у цьому середовищі та як цим користуватися. Гайди з налаштування — у [[50-Guides/Гайди|50-Guides]].

## Підключені конектори (станом на 2026-06-11)

| Конектор | Можливості |
|---|---|
| GitHub (плагін engineering) | Робота з репозиторіями, PR, issues |
| Gmail | Пошук листів, чернетки, мітки |
| Notion | Сторінки, бази даних |
| Figma | Читання/створення дизайнів, діаграми FigJam |
| Google Drive | Пошук і читання файлів |
| Slack, Linear, Jira/Confluence, Datadog, PagerDuty, Asana | Доступні через плагін engineering (потребують автентифікації) |
| Claude in Chrome | Керування браузером |
| Computer use | Керування десктопними застосунками |

## Навички (skills) від Anthropic

| Навичка | Для чого |
|---|---|
| docx / pdf / pptx / xlsx | Створення й редагування документів |
| engineering:architecture | ADR — архітектурні рішення |
| engineering:system-design | Проєктування систем (знадобиться для NMDB) |
| engineering:code-review | Рев'ю коду перед merge |
| engineering:documentation | Технічна документація, README, runbook |
| engineering:testing-strategy | Стратегії тестування |
| schedule | Заплановані задачі (щоденні звіти тощо) |
| skill-creator | Створення власних навичок |
| mcp-builder | Створення MCP-серверів (для NMDB) |

## Власні навички середовища (`.claude/skills/`)

| Навичка | Тригер | Що робить |
|---|---|---|
| `new-project` | «створи проєкт <Назва>» | Папка в `10-Projects/` з Огляд/Вимоги/План/Контекст + реєстрація в індексах |
| `save-context` | «збережи контекст» (перед /clear) | Оновлює `<Назва> — Контекст.md`, чекбокси плану, ADR, коміт |
| `weekly-review` | «розбери вхідні», «огляд тижня» | Розбір 00-Inbox, перевірка контекстів і посилань, архівація |

Команди-кнопки до них — у `.claude/commands/` (`/new-project`, `/save-context`, `/weekly-review`), див. [[50-Guides/Гайд — Команди для агентів]].

Формат — за практиками Anthropic ([[20-Knowledge/AI OS — практики Four Cs|деталі]]): коротка SKILL.md, «пушна» description з тригерами, правило + чому.
