# Claude

**Конфіг:** `CLAUDE.md` (корінь) → посилається на `AGENTS.md`. Папка `.claude/` — для локальних налаштувань Claude Code (settings.json, навички проєкту).

## Для чого використовувати

- Робота зі сховищем: структурування нотаток, дашборди, плани
- Архітектура та документація проєктів (DBManager тощо)
- Генерація документів: docx, pdf, pptx, xlsx (через навички)
- Код: рев'ю, рефакторинг, написання модулів
- Автоматизація: заплановані задачі, конектори (Gmail, Notion, GitHub, Figma…)

## Особливості

- У Cowork має прямий доступ до файлів цієї папки та ізольованого Linux-терміналу.
- Навички (skills) підключаються в Settings → Capabilities.
- Пам'ять між сесіями — персистентна (запам'ятовує контекст проєктів).

## Корисні команди Claude Code (термінал)

```
claude            # запустити в C:\Dev\Meweek
/init             # згенерувати/оновити CLAUDE.md з аналізу папки
/memory           # редагувати пам'ять
```
