## 🤖 Claude Code та агенти

**Ключовий патерн тижня — Harness Engineering (Борис Черні):**
Продуктивна AI-розробка = не окремі промпти, а *цикли* (loops): агент спостерігає → планує → діє → рефлексує. Програма-оболонка запускає агента за cron, передає контекст, обробляє помилки.

**Вкладені sub-agents (до 5 рівнів)** відкривають нові архітектурні можливості:
- Orchestrator → спеціалізовані агенти → мікро-агенти для підзадач
- Паралельне виконання на спільній файловій системі
- Managed Agents від Anthropic беруть на себе orchestration, sandboxing, session state

**Dreaming** — новий scheduled процес Claude Code: переглядає сесії агентів, виявляє патерни, оновлює memories. Агенти покращуються самостійно між сесіями.

## Для Meweek

**1. Що варто спробувати сьогодні?**
Активувати Agent SDK кредити до 15 червня + протестувати `--safe-mode` для дебагу середовища.

**2. Яку нову навичку додати в `.claude/skills`?**
`harness-loop` — навичка для запуску агентного циклу через `claude -p` з cron (harness engineering патерн від Борisa Черні).

**3. Яку рутину можна автоматизувати?**
Цей самий дайджест — `claude -p` задача за cron о 07:00, якщо до 15 червня, або через Agent SDK кредити після.

**4. Який MCP сервер заслуговує уваги?**
GitHub MCP (вже є) + Filesystem MCP для повнішого доступу до Meweek-файлів без ручного `/read`.

**5. Яка одна ідея сьогодні дасть найбільшу економію часу?**
Активація Agent SDK кредитів — без цього всі `claude -p` автоматизації зупиняться 15 червня.
