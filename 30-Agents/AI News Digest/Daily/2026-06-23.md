### 1. Claude Code Dynamic Workflows (Ultracode) — GA
Claude Code офіційно вивів Dynamic Workflows у загальний доступ. Тригер — ключове слово `ultracode` у промпті. Claude автоматично планує задачу, розбиває на підзадачі, запускає їх паралельно у субагентах і зводить результат. Попередження: витрати токенів суттєво вищі, починати з невеликих задач.

### 4. Loop Engineering — наступний рівень після Prompt Engineering
Борис Черні (голова Claude Code): «Я більше не пишу промпти — Claude пише промпт, а я говорю з цим новим Claude, що координує». Сформувався термін **Loop Engineering** — людина задає цілі й контролює цикл, а не пише кожен запит вручну.

**Пріоритет**: 🔴 | **Impact**: High | **Effort**: Medium | **Confidence**: High
**Статус**: 🟡 Review First

### 5. Karpathy — Idea File замість коду
У новому твіті Karpathy описав концепцію **Idea File**: у епоху LLM-агентів ділитись треба ідеями (абстрактний опис + мета), а не кодом. Агент одержувача сам будує реалізацію під його інструменти та контекст.

## Для Meweek

**1. Що варто спробувати сьогодні?**
Ввести `ultracode` у наступний великий запит (наприклад, `/weekly-review`) і подивитися, як Claude будує workflow з субагентами.

**2. Яку нову навичку додати в `.claude/skills`?**
`ultracode-workflow` — обгортка, що автоматично додає ключове слово `ultracode` до важких задач і логує витрати токенів.

**3. Яку рутину можна автоматизувати?**
Claude Managed Agents Vaults — перенести API-ключі рутин у вaults і увімкнути автоматичний розклад безпосередньо в хмарі замість ручного cron.

**4. Який MCP сервер заслуговує уваги?**
Playwright MCP — Anthropic використовує його для автоматичного UI-тестування у своєму harness. Корисний для верифікації змін у веб-проєктах.

**5. Яка одна ідея дасть найбільшу економію часу?**
`ultracode` у `/weekly-review` — замість лінійного перегляду Claude паралельно обробляє кілька розділів сховища і зводить результат.
