## 🤖 Claude Code та агенти

**Ключові інсайти (15–24 червня):**

1. **`/rewind` — критична функція для рутин**: рутини типу daily-digest можуть тепер відновлювати контекст після `/clear` без перезапуску сесії.
2. **`sandbox.credentials` — security best practice**: для будь-якого CI/CD або автоматичного запуску Claude Code — додати в `.claude/settings.json`.
3. **5-рівневі subagents**: відкриває складні оркестровані pipeline, але >3 рівні важко відлагоджувати — `/agents` команда показує дерево.
4. **CPU -37%** під час стримінгу — оновлення варте само по собі при тривалих сесіях.
5. **Claude Tag паттерн**: async агент у комунікаційному каналі = нова операційна модель, де агент бачить весь командний контекст, а не лише свій тред.

## Для Meweek

**1. Що варто спробувати сьогодні?**
Додати `"sandbox": {"credentials": true}` і `"fallbackModel": ["claude-sonnet-4-6", "claude-haiku-4-5-20251001"]` у `.claude/settings.json` — два параметри, які одразу покращують безпеку і надійність всіх рутин.

**2. Яку нову навичку додати в `.claude/skills`?**
`ai-os-audit` — щотижневий аудит AI OS: які задачі повторювались? що не автоматизовано? яка навичка деградує? Натхнення: Nate Herk's `/audit` з AIS-OS.

**3. Яку рутину можна автоматизувати?**
DeepSeek endpoint migration check — мала рутина перевіряє чи є в проєктах Meweek інтеграції з `deepseek-chat`/`deepseek-reasoner` і нагадує про дедлайн 24 липня.

**4. Який MCP сервер заслуговує уваги?**
Enterprise managed connector через Okta — якщо Meweek масштабується до команди, це готова zero-touch provisioning інфраструктура для MCP. Зараз — стежити.

**5. Яка одна ідея сьогодні дасть найбільшу економію часу?**
`/rewind` — якщо рутина впала на середині, тепер не треба перезапускати всю сесію. Перевірити в наступній сесії з `/clear`.
