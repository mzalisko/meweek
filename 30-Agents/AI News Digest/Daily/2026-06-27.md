## Для Meweek

**1. Що варто спробувати сьогодні?**
`/rewind` — якщо після `/clear` потрібен контекст назад. Перевірити чи доступно у поточній версії Claude Code.

**2. Яку нову навичку додати в `.claude/skills`?**
`claude-tag-workflow` — навичка для організації мультиагентної роботи в Slack-стилі: один @Claude на задачу, проактивний follow-up, розбивка на етапи.

**3. Яку рутину можна автоматизувати?**
MCP-сервер authentication — зараз треба вручну, тепер `claude mcp login` можна вбудувати в startup hook.

**4. Який MCP сервер заслуговує уваги?**
`claude-obsidian` (12,900+ зірок, офіційна підтримка Steph Ango) — дає агентам traverse wikilink-граф Meweek без ручного `Read` кожного файлу.

**5. Яка одна ідея сьогодні дасть найбільшу економію часу?**
**`/rewind`** — якщо контекст загубився після /clear, не треба перечитувати файли вручну. Одна команда відновлює сесію.
