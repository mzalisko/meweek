### 2. 🔴 Claude Code: /rewind, OpenTelemetry, автодоповнення шляхів
Нові фічі в останніх релізах (кінець червня):

## 🚀 Нові практики для впровадження

### Loop Engineering — писати цикли замість промптів
- **Пріоритет**: 🔴 Critical
- **Impact**: High | **Effort**: Medium | **Confidence**: High
- **Статус**: 🟡 Review First (вивчити → потім впроваджувати)
- **Чому важливо**: це наступний рівень після single-agent промптингу. Цикл = автономна система, що сама вирішує коли і як запускати агентів. Boris Cherny і Anthropic вже живуть так 8 місяців.
- **Де fits**: weekly-review, daily-digest, fitness звіти, code-review рутини.
- **Наступний крок**: прочитати `Harness Engineering — агентні цикли.md` + статтю Cherny про loops. Написати перший цикл для daily-digest.

### OpenTelemetry для observability агентних систем
- **Пріоритет**: 🟡 Useful
- **Impact**: Medium | **Effort**: Low | **Confidence**: High
- **Статус**: 🟢 Ready to Apply
- **Чому важливо**: без трейсингу не видно, де агент ухвалює хибні рішення. Новий лог-подія в Claude Code дає текст відповіді моделі у структурованому форматі.
- **Де fits**: будь-яка рутина, що запускає більше 3 агентів послідовно.
- **Наступний крок**: увімкнути `CLAUDE_CODE_OTEL_EXPORTER_OTLP_ENDPOINT` у тестовій рутині і переглянути трейси.

## Для Meweek

**1. Що варто спробувати сьогодні?**
Замінити `/clear` на `/rewind` у щоденній роботі — одна нова звичка, нульовий effort, зберігає контекст сесій без ручного збереження.

**2. Яку нову навичку додати в `.claude/skills`?**
`loop-engineering` — шаблон для написання агентних циклів: вхідні умови, логіка повторень, вихідні умови, вбудований budget-контроль. Замість одноразових промптів — перевикористовувані автономні цикли.

**3. Яку рутину можна автоматизувати?**
Моніторинг Claude Code changelog: щоденно порівнювати поточну версію з попередньою, виявляти нові фічі та автоматично оновлювати відповідний файл у `20-Knowledge/AI Practices/`.

**4. Який MCP сервер заслуговує уваги?**
**Obsilo Agent** для Obsidian — 55+ інструментів + 3-tier memory + MCP connectors. Якщо справді працює, це замінить ручне `Read` файлів нотаток і дасть агентам семантичний контекст по всьому сховищу Meweek.

**5. Яка одна ідея сьогодні дасть найбільшу економію часу?**
Loop Engineering для `daily-digest` рутини: замість кожного разу запускати агента з нуля — написати цикл, що сам визначає джерела, збирає новини, фільтрує дублі і форматує звіт. Одноразова інвестиція → постійна економія щодня.
