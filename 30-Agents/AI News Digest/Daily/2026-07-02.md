## 🤖 Claude Code та агенти

1. **Фонові агенти стали автономнішими**: авто-commit/push/draft PR по завершенню — це саме та поведінка, за якою зараз працює й ця сесія (хмарна рутина `daily-ai-digest`).

## Для Meweek

**1. Що варто спробувати сьогодні?**
Перевірити, чи Fable 5 знову доступний без обмежень у моделях для рутин — раніше міг бути позначений як недоступний.

**2. Яку нову навичку додати в `.claude/skills`?**
Сьогодні немає достатньо сильної ідеї для нової навички — знахідки зафіксовано в нотатках знань замість пропозиції скіла.

**3. Яку рутину можна автоматизувати?**
Підключити Notification hooks (`agent_needs_input`/`agent_completed`) до фонових рутин замість ручної логіки сповіщень у промпті.

**4. Який MCP сервер заслуговує уваги?**
SnapLogic MCP Builder — якщо колись знадобиться швидко обгорнути існуючу інтеграцію в MCP-сервер без коду. Не терміново.

**5. Яка одна ідея сьогодні дасть найбільшу економію часу?**
Спробувати `/dataviz` skill для звітів Fitness/weekly-review — вже встановлений, просто не використовується.
