## 🔴 Найважливіше

1. **Loop Engineering — новий парадигм агентної роботи** (Boris Cherny / Fortune)
   Черні пояснив: він більше не пише промпти — Claude пише їх сам, а Черні розмовляє з тим Claude, що координує. Вірусна дискусія у червні сформулювала концепцію "loop engineering": агентна оркестрація як повторювана системна задача, а не діалог хід-за-ходом. Черні керує від сотень до десятків тисяч агентів на день.

2. **Claude Code авто-режим тепер блокує деструктивні git-команди** (v2.1.183, 19.06)
   `git reset --hard`, `git checkout -- .`, `git clean -fd`, `git stash drop` заблоковані без явного запиту. `git commit --amend` — якщо коміт не зроблений агентом цієї сесії. Terraform/Pulumi/CDK destroy — без явного стека. Прямий вплив на безпеку Meweek-рутин.

3. **Dynamic Workflows — сотні паралельних субагентів з одного сценарію** (Code with Claude, Week 22)
   Claude Code пише скрипт оркестрації і запускає десятки-сотні субагентів паралельно. Використано для міграцій кодових баз у сотні тисяч рядків. Доступно на Enterprise/Team/Max.

4. **MCP Apps — інтерактивні UI у відповідях агентів** (MCP специфікація)
   Спільна розробка Anthropic + OpenAI. Інструменти повертають HTML-інтерфейс у sandbox iframe прямо в чаті. Підтримується Claude, ChatGPT, Goose, VS Code. Партнери: Figma, Slack, Salesforce, Canva, Asana, Amplitude.

5. **OpenAI придбала Astral** (uv + ruff)
   Два найпопулярніші Python-інструменти 2025-2026 (замінники pip/black/flake8) переходять під контроль OpenAI. Інтеграція в Codex. Ризик для незалежності екосистеми Python.

## Для Meweek

**1. Що варто спробувати сьогодні?**
Ввімкнути `/config autoMode=true` у тестовій Meweek-сесії — переконатись, що деструктивні git-команди заблоковані.

**2. Яку нову навичку додати в `.claude/skills`?**
`loop-engineer` — навичка перетворення одиничного промпту у багаторівневий агентний цикл (координатор → спеціалізовані субагенти).

**3. Яку рутину можна автоматизувати?**
`weekly-review` — переписати як Loop: координатор аналізує зміни за тиждень → запускає субагентів per-project → збирає підсумки.

**4. Який MCP сервер заслуговує уваги?**
**Obsidian Smart Connections MCP** — дозволяє агентам читати і записувати у сховище без CLI. Перевірити сумісність із Meweek-структурою.

**5. Яка одна ідея сьогодні дасть найбільшу економію часу?**
`Tool(param:value)` правила для моделей — один рядок у `.claude/settings.json` і дорогі субагенти (Opus) обмежуються тільки там, де потрібна висока якість.
