# AI Дайджест 2026-06-28

## 🔴 Найважливіше

1. **OpenAI анонсував GPT-5.6 Sol/Terra/Luna** — нове сімейство моделей прем'єрного рівня. Sol — frontier reasoning + довгострокові агентні задачі, Terra — баланс ціна/якість (2x дешевше за GPT-5.5), Luna — найшвидший і найдешевший. Запускається «ultra» режим із паралельними субагентами. Обмежений доступ (~20 компаній) — очікується широкий rollout наступного тижня.

2. **Anthropic vs Alibaba — найбільша відома атака дистиляції** — Anthropic звинуватив Alibaba у 28.8 млн обмінів з моделями через ~25 000 шахрайських акаунтів (22 квітня — 5 червня). Це не теоретична загроза, а задокументована промислова атака на закриті моделі.

3. **Karpathy: AI стає "постійним тиммейтом"** — 3-я ера UX для LLM: не чатбот, не асистент, а персистентна асинхронна сутність, вбудована в робоче середовище організації. «Software 3.0» — автоматизує все, що можна верифікувати. Karpathy тепер у команді Anthropic (pretraining).

4. **Claude Code Week 26** — `claude mcp login` (автентифікація MCP з шелу), `/rewind` (повернення до стану до `/clear`), shell mode пояснює вивід команд без додаткового промпту, субагенти тепер виводять permission prompts у головну сесію замість auto-deny.

5. **MCP 2026-07-28 Release Candidate** — найбільша ревізія протоколу від запуску: stateless core (звичайний HTTP load balancer), MCP Apps (server-rendered UI), Tasks extension (довготривалі задачі), оновлений OAuth/OIDC. Фінальний spec — 28 липня.

- **Наступний крок**: Використати в наступній сесії weekly-review замість `/clear`.

## Для Meweek

**1. Що варто спробувати сьогодні?**
`/rewind` у поточній або наступній сесії — повернення контексту без перезапуску.

**2. Яку нову навичку додати в `.claude/skills`?**
`mcp-auth` — автоматизація `claude mcp login` для всіх налаштованих серверів; або `workflow-starter` — шаблон dynamic workflow для рутин Meweek.

**3. Яку рутину можна автоматизувати?**
Цей самий дайджест через Dynamic Workflows — зробити більш паралельним: 5-7 агентів одночасно шукають по різних джерелах, 1 зводить результат.

**4. Який MCP сервер заслуговує уваги?**
**MCP Tasks extension** (у RC) — для довготривалих асинхронних задач. Коли стане доступним, зможе замінити поточний підхід з poll-циклами.

**5. Яка одна ідея сьогодні дасть найбільшу економію часу?**
`fallbackModel` у settings.json — 5 хвилин налаштування, захищає всі автоматичні рутини від збоїв через ліміти моделей.
