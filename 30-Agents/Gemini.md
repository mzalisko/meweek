# Gemini

**Конфіг:** `GEMINI.md` (корінь) → посилається на `AGENTS.md`. Папка `.gemini/` — локальні налаштування (settings.json); можна ввімкнути читання `AGENTS.md` напряму через `"contextFileName": "AGENTS.md"`.

## Встановлення

```
npm install -g @google/gemini-cli
gemini           # запустити в C:\Dev\Meweek
```

## Для чого використовувати

- Задачі з дуже великим контекстом (аналіз великих кодових баз, довгих документів)
- Альтернативна думка щодо архітектурних рішень
- Безкоштовний ліміт для рутинних задач
