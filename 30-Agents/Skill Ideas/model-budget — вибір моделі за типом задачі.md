# Пропозиція навички: model-budget

## Назва навички
`model-budget`

## Мета

Допомогти вибрати оптимальну модель Claude для кожного типу задачі в Meweek — баланс між якістю результату та витратою кредитів з Agent SDK пулу. Особливо актуально після 15 червня 2026, коли `claude -p` рутини витрачають кредити за API-тарифами.

## Вхідні дані
- Тип задачі (опис або категорія)
- Опціонально: бюджетний пріоритет (економія / баланс / якість)
- Опціонально: очікуваний обсяг виводу (малий / середній / великий)

## Вихідні дані
- Рекомендована модель з обґрунтуванням
- Орієнтована вартість ($/1K токенів виводу)
- Приклад `--model` флагу або `availableModels` конфіга
- Попередження якщо задача не відповідає вибраній моделі

## Матриця вибору моделі

| Тип задачі | Рекомендована модель | Причина |
|---|---|---|
| Автоматичні рутини (дайджест, weekly review) | `claude-sonnet-4-6` | Якість достатня, ~5× дешевше Fable |
| Масова обробка, прості трансформації | `claude-haiku-4-5` | Найшвидше і найдешевше |
| Fitness звіти (шаблонна генерація) | `claude-haiku-4-5` | Структурований вивід, мінімум міркування |
| Архітектурні рішення, складний аналіз | `claude-fable-5` | Потрібна максимальна якість |
| Інтерактивна розробка | Default (Fable 5) | Максимальні можливості для роботи в парі |
| Code review, security review | `claude-sonnet-4-6` | Баланс якості та вартості |

## Приклад воркфлоу

```
/model-budget "написати щоденний дайджест новин"
→ Аналізує тип задачі
→ Виводить: "Рекомендую: claude-sonnet-4-6"
→ Причина: "Шаблонна задача, достатньо середньої складності моделі"
→ Вартість: "~$0.015 за звіт (3K output токенів × $5/M)"
→ Порівняння: "Fable 5 коштував би ~$0.15 — 10× дорожче за той же результат"
→ Конфіг: 'Додати до SKILL.md: availableModels: ["claude-sonnet-4-6"]'
```

## Оцінка цінності

- **Висока**: з `claude -p` пулом $20/міс (Pro) різниця між Fable і Sonnet — це 133 vs 1333+ запусків на місяць
- Одноразовий аудит може заощадити 80-90% кредитів без втрати якості результату

## Складність реалізації

- **Низька**: навичка-консультант, не потребує зовнішніх API
- Потрібна підтримка актуальної матриці моделей та цін (оновлювати при зміні лінійки)
