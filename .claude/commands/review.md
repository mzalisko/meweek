Проведи критичне рев'ю поточних незакомічених змін (git diff + git status) як незалежний рев'ювер.

Перевір по черзі:
1. Безпека: ін'єкції, XSS, секрети в коді/нотатках, надлишкові права (для NMDB — звірити з 10-Projects/NMDB/NMDB — Безпека.md).
2. Коректність: edge cases, обробка помилок, що станеться при недоступності залежностей.
3. Відповідність правилам AGENTS.md (мова, структура, посилання).

Формат висновку: список зауважень за пріоритетом (блокер / варто виправити / дрібниця), кожне з файлом і рядком. Якщо зауважень нема — явно написати, що перевірено і чисто. НЕ виправляй нічого без підтвердження користувача.

Додатковий фокус рев'ю (якщо вказано): $ARGUMENTS
