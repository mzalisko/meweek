# Щотижневий огляд

## Кроки

1. Розбери `00-Inbox/`: кожну нотатку перенеси у проєкт / Базу знань / Ресурси або запропонуй видалити.
2. Перевір кожен активний проєкт у `10-Projects/Проєкти.md`: чи актуальні `— Контекст.md` і статуси.
3. Перевір биті посилання `[[...]]` по сховищу.
4. Завершені проєкти запропонуй перенести в `90-Archive/`.
5. Підсумок — у щоденну нотатку `00-Inbox/Journal/`, коміт: `огляд: тижневий розбір сховища`.
