# 🏠 Meweek — головний дашборд

Це центральна точка входу до сховища. Все середовище живе в `C:\Dev\Meweek` і синхронізується через git ([репозиторій](https://github.com/mzalisko/meweek)).

## Активні проєкти

- [[10-Projects/DBManager/DBManager — Огляд|DBManager]] — централізована система даних для WP-сайтів (DataBridge)
- [[10-Projects/Fitness/Fitness — Огляд|Fitness]] — персональний проєкт схуднення, меню, тренувань і журналу прогресу

## Швидкі дії

- Нова ідея → кинути в `00-Inbox`
- Новий проєкт → команда `/new-project` або «створи проєкт <Назва>»
- Зберегти контекст перед /clear → `/save-context` або «збережи контекст»
- Технічне рішення → оформити через [[_Templates/Шаблон ADR]]
- Людські гайди → [[50-Guides/Гайди]]
- Правила для агентів → [[30-Agents/Гайди для агентів/Гайди для агентів]]

## Правила ведення

1. Вміст нотаток — **українською**; назви папок — англійською.
2. Контекст для AI-агентів живе у файлах `<Проєкт> — Контекст.md` — оновлювати перед `/clear` (див. [[30-Agents/Робота з контекстом]]).
3. Кожен проєкт має файл `<Назва> — Огляд.md` — це його дашборд.
4. Вкладення (скріншоти, файли) автоматично падають у `40-Resources/Attachments`.
5. Граф Obsidian покращувати через `[[wiki-посилання]]`, а не ручну косметику `.obsidian/`.
6. Раз на тиждень: розібрати `00-Inbox` (`/weekly-review`), закомітити зміни в git.
