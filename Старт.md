# 🏠 Meweek — головний дашборд

Це центральна точка входу до сховища. Все середовище живе в `C:\Dev\Meweek` і синхронізується через git ([репозиторій](https://github.com/mzalisko/meweek)).

## Розділи сховища

| Розділ | Призначення |
|---|---|
| [[00-Вхідні/Вхідні\|00-Вхідні]] | Швидкі нотатки, ідеї, щоденник. Все, що ще не розкладено. |
| [[10-Проєкти/Проєкти\|10-Проєкти]] | Активні проєкти. Кожен проєкт — окрема папка зі своїм дашбордом. |
| [[20-База знань/База знань\|20-База знань]] | Постійні знання: технології, рішення, how-to. |
| [[30-Агенти/Агенти\|30-Агенти]] | AI-агенти (Claude, Codex, Gemini), їх правила, навички, конектори. |
| [[40-Ресурси/Ресурси\|40-Ресурси]] | Посилання, вкладення, зовнішні матеріали. |
| 90-Архів | Завершені проєкти й застарілі нотатки. |
| _Шаблони | Шаблони нотаток (проєкт, зустріч, ADR, щоденник). |

## Активні проєкти

- [[10-Проєкти/NMDB/NMDB — Огляд|NMDB]] — централізована система даних для WP-сайтів (DataBridge)

## Швидкі дії

- Нова ідея → кинути в `00-Вхідні`
- Новий проєкт → скопіювати [[_Шаблони/Шаблон проєкту]] у `10-Проєкти/<Назва>/`
- Технічне рішення → оформити через [[_Шаблони/Шаблон ADR]]

## Правила ведення

1. Усі нотатки — **українською мовою**.
2. Кожен проєкт має файл `<Назва> — Огляд.md` — це його дашборд.
3. Вкладення (скріншоти, файли) автоматично падають у `40-Ресурси/Вкладення`.
4. Раз на тиждень: розібрати `00-Вхідні`, закомітити зміни в git.
