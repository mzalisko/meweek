# NMDB — План (чернетка)

⚠️ Попередній план верхнього рівня. Детальне планування — після відповідей на питання з [[NMDB — Вимоги]].

Проєкт: [[NMDB — Огляд]]

## Етапи

### Етап 0 — Уточнення (поточний)

- [ ] Відповісти на питання з [[NMDB — Вимоги]]
- [ ] Обрати стек і хостинг
- [ ] ADR: архітектура (центр / DataBridge / плагін), модель доставки даних (push/pull)

### Етап 1 — Прототип

- [ ] Мінімальна центральна система: CRUD значень + проста адмінка
- [ ] DataBridge: read-only API з токенами
- [ ] WP-плагін: отримання даних, кеш, шорткод для виводу
- [ ] Перевірка сценарію «DataBridge недоступний → сайт працює з кешу»

### Етап 2 — Безпека та стабільність

- [ ] Реалізувати вимоги з [[NMDB — Безпека]] (HMAC, ротація токенів, rate limit, журналювання)
- [ ] Безпековий рев'ю коду
- [ ] Бекапи та відновлення

### Етап 3 — Продакшн

- [ ] Розгортання на бойовому хостингу
- [ ] Підключення першого сайту
- [ ] Моніторинг, документація (runbook)

### Етап 4 — Розширення

- [ ] Кілька сайтів, ролі користувачів, історія змін
- [ ] Блоки Gutenberg / віджети у плагіні
